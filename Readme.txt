作品名　 :フィーナの風旅
ジャンル :パズルアクションゲーム
作品の概要
	このゲームは風の力をうまく利用して主人公「フィーナ」を
	ある島まで導いてあげるゲームです。
	2種類の風車をうまく用いて旅路にある障害物をうまく避けましょう。
	さて、フィーナは無事に島に辿り着けるのだろうか…

アピールポイント
	僕らのゲームは、色々な所にこだわって制作しましたが、
	特にこだわったのは、風の挙動です。
	よりリアルに近い風を作ろうと、風車を置いた時や送風機を
	置いた時の風の流れをうまく表現しました。
	それと主人公「フィーナ」はチームで決めたキャラクターなので
	ぜひエンディングで可愛いフィーナを見てもらいたいです。
	
	フィーナの風旅.movのムービーにまとめてあります。
	
起動方法
	フィーナの風旅.exeとdataフォルダを同じ階層のパスを置きフィーナの風旅.exeを起動してください。

操作方法
	キーボード入力でのプレイ出す。
	矢印キー、Enterキー、X、Z、Yキーを使用します。
	詳細は、操作説明.pdfをご覧ください。
	
	ゲームパッドでのプレイ対応
	左ジョイスティックor矢印、○、△、×、R1を使用します。
